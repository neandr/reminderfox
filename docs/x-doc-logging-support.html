JavaScript Console Logging in Reminderfox

A new Logger is used with the Reminderfox CalDAV implementation. The JS code has 'Logs' to be used to pass message strings to the console output and a 'level' system is used to individually control the output.Optionally the STACK and filename/line number can be appended to the output.

To enable/configure these features the Firefox/Thunderbird preference system is used, the prefs parameter is "extensions.reminderFox.loggers", it's a string type with JSON notation.

The general setup is: extensions.reminderFox.loggers ; {rootLogger:level, subLogger:level}

Each logging is set with a pair of “logger“:”level“ (not case sensitive), multiple logging details are separated by comma.

"level" values can be any of  'Fatal', 'Error', 'Warn', 'Info', 'Config', 'Debug', 'Trace', 'All'. ('Fatal' is highest, 'All' is lowest level).

A special logger is the rootLogger / 'Reminderfox' element, without it the whole logging is disabled.

SubLoggers are defined also with name/level:

Examples:     {"Reminderfox":"Error","calDAV":"Info"}     {"Reminderfox":"Trace","calDAV":"Info"}

The Logger will send an output to the console if a 'subLogger' has the same or a higher "level" than 'Reminderfox'. ad Examples above:

first   will NOT send the message, second  will send.

 

To use the Logger a function call has to be added to the JS code: reminderfox.util.Logger ('calDAV', msg);

Call to Logger will not send anything to the console with -- prefs string not defined -- prefs string without 'Reminderfox' and 'level' -- call with unknown 'level' for 'subLogger'

Additional console output -- for 'Fatal', 'Error', 'Warn': the calling STACK will be appended to the normal log message -- with 'Info' or higher: the calling filename and line number is appended.

Using a call for the Logger with 'Alert' will write to console independent of 'Reminderfox' settings!

'Alert' can have different writings and writes to the console

- 'Alert' :   message only

- 'ALERT' : message and call stack with depth of 10

- 'alert' :  write message and call stack with depth of 1
