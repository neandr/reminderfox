2. CalDAV Testing

Enable the following Firefox/Thunderbird preference to add an extra tab to the Options -->  [ Sync ]
preference name value description
extensions.reminderFox.debug.CalDAVtest true enable TestTab in Options --> Sync

The first part is to define the Request Use the Calendar menu to select one of the CalDAV accounts defined on [CalDAV Calendars].

Select a predefined method (like 'getCTAG'), that will fill in the method (like 'PROPFIND') and the header ('Depth: 0'). 'urlstr' is for advanced requests (e.g. send a new event) and combines the url and the event UID. The icon is used to copy the request to the clipboard.

Result will hold the CalDAV return values The filter box can be used to post-process the Result. Icons are used to clear and also copy to the clipboard. The left icon recalls the last complete result.
