ReminderFox deinstallieren

Warum wollen sie wohl so etwas machen? OK, falls so ein Wunsch bestehen sollte, so kann die Erweiterung ganz einfach mit dem Firefox oder Thunderbird Erweiterungsmanager deinstalliert werden. Hierzu auf der Hauptmenüliste von Firefox/Thunderbird das Menü [Extra] --> [Erweiterungen] wählen. Dort ReminderFox aktivieren und die Schaltfläche [Entfernen] auswählen. Nach dem nächsten Start von Firefox/Thunderbird wird ReminderFox entgültigt entfernt. Schließlich kann noch die Datei mit den Erinnerungen gelöscht werden. Diese liegt normalerweise im Profilverzeichnis, ggf. vor der ReminderFox Deinstallation über ReminderFox Einstellungen und dort auf dem Tab "Lokale Datei" nachsehen wo die Datei liegt.

C:\Documents and Settings\Administrator\Application Data\Mozilla\Firefox\Profiles\qf5aadhx.default\reminderfox
