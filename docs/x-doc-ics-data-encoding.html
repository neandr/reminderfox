ICS Data Editing -- Note about Encoding

ICS Calendar data with Reminderfox is stored in text files with .ics filename extension.

So from a general view those files can be edited with text editors. The important point is

ICS Calendar data has to is be stored with UTF-8 encoding ... ALWAYS

Some operating systems are set with other encoding (ANSI, Unicode etc). So using a standard editor which doesn't support UTF-8 or if the used editor isn't set for UTF-8 encoding this will harm ICS data.

An ICS file written with wrong encoding will not produce correct reminders/todos!

Windows - Notepad

A common used editor in the Microsoft world is the Notepad. Make sure to set the encoding for UTF-8 as shown below.
