Calendar Data Import

Reminderfox is based on the industry standard format (RFC) which is widely used with calendar systems like Lightning (Thunderbird), Google Calendar, Apple iCal. This let the user interchange data between different calendar systems.

Also the internet provides numerous sources of different events calendars for holidays, sports or culture events.

Calendar data is also send by email with travel booking like flight or train reservations.

The import of those calendar data to Reminderfox can be made easily with different methods directly to the Reminderfox calendar, to the current used or also a new calendar can be defined.

Import Dialog

The import is started on the source of the data with drag&drop or with a context menu (see below). All methods open a dialog to allow different import methods:

The first part informs about the data selected for import. The second part starts with "How to handle the Import?" and with showing the three provided import methods:

  • Add -- Imported Events and ToDos will be added to the existing Reminderfox calendar. Imported duplicates can be ignored or only newer will be imported.
  • New -- Imported Events and ToDos will be used to build a new Calendar.Â The existing Calendar will be closed and saved. The calendar data is stored in the same directory as the 'existing' was stored.
  • Subscribe -- Imported events builds a Reminderfox User List which will be linked to the external data provider. Changes on the external page will be shown on Reminderfox list with each opening of the list. With this method the events on the Reminderfox Main Dialog are disabled. For individual usage/modification single events can be copied to the 'Reminders'
